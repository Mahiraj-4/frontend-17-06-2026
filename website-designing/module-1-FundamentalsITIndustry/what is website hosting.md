# What is website hosting
Web hosting is a service that securely stores your website's files (code, images, videos) on a specialized server. This makes your site accessible 24/7 to anyone on the internet. Think of it like renting physical land on the internet to build your "house" (website).

# how it works
When a user types your domain name (e.g., yourwebsite.com) into their web browser, the internet routes their request to the server housing your files. The server acts as a delivery system, sending the necessary website data back to the user's device so they can view the webpage.

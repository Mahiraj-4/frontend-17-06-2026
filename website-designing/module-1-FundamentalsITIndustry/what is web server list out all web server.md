# What is web server
A web server is software and underlying hardware that stores website files (HTML, images, CSS) and delivers them to users over the internet via HTTP or HTTPS. Its primary job is to accept requests from web browsers and serve the requested web pages.
While there are dozens of niche and proprietary servers, the landscape is largely dominated by a few major players. Here are the most widely used web servers today.

# List out all web server
Top web servers
1. Nginx:
Currently the market-leading web server globally. Known for its extremely high performance, low memory usage, and ability to handle thousands of concurrent connections easily.

2. Apache HTTP Server:
One of the oldest and most reliable web servers. It is highly customizable and powers a vast portion of the internet.

3. LiteSpeed: 
A high-performance, drop-in replacement for Apache. It is highly popular in commercial web hosting for its speed and efficient handling of traffic spikes.

4. Microsoft IIS (Internet Information Services):
Microsoft's proprietary web server designed for Windows Server environments.

Other Notable Web Servers
1. Node.js:
While primarily a runtime environment, it includes built-in modules that allow it to act as its own web server, commonly used for real-time applications.

2. Apache Tomcat:
A specialized Java-focused web server and servlet container that runs Java code.

3. Caddy:
A modern, beginner-friendly web server that automatically handles HTTPS encryption (SSL/TLS).

4. OpenLiteSpeed:
The open-source edition of the LiteSpeed web server.

5. Lighttpd: 
An optimized web server designed for speed-critical environments with very low memory consumption.

6. Eclipse Jetty:
Commonly used in machine-to-machine communications and often embedded in larger applications.

7. H2O:
An optimized web server designed for HTTP/2.

8. Hiawatha:
A web server focused heavily on security and ease of use.

9. Legacy/Enterprise:
Systems like IBM WebSphere and older Netscape servers are still found in some legacy corporate setups.

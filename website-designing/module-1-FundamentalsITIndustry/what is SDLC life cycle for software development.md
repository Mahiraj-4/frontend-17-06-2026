# What is SDLC life cycle for software development
The Software Development Life Cycle (SDLC) is a structured, step-by-step methodology used by teams to plan, create, test, and deploy high-quality software. It provides a systematic framework to minimize risks, control costs, and ensure the final product meets user and business requirements.
The SDLC is typically broken down into seven core phases. While models like Agile allow teams to overlap and repeat these phases quickly, others like Waterfall require them to happen in a strict, sequential order.

1. Planning and Analysis
2. Requirements Gathering
3. System Design
4. Development
5. Testing
6. Deployment
7. Maintenance
